# Facebook Extension

This project contains the browser extension source originally stored in `chrome extention`.

## Contents
- `manifest.json` — Extension manifest
- `background.js` — Background script
- `contentScript.js` — Content script for page modifications
- `myStyles.css` — Custom styles injected into the target page

## Notes
- The extension was renamed to `facebook-extension` for clarity.
- Confirm compatibility with the current Facebook Business Suite version.
